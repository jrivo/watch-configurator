# watch-configurator


### json example

```json
{
	type: "watch",
	collection: ["Steel", "Gold"]
	dial: ["black", "white", "green"]
	strap: [
			{
				material: "rubber_strap",
				style: "thin" || "large",
				color: "black" || "white" || "green",
			},
			{
				material: "leather_strap"
				style: "thin" //always thin
				color: "black" || "white" || "green"
			}
		]
	length: ["S-M", "L-XL"]
}
```
